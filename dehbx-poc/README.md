# De-HugoBlox POC

This folder is a safe sandbox to test a fully independent site (no Hugo, no HugoBlox runtime).

## Run locally

From this folder:

```powershell
node .\server.js
```

Then open:

`http://localhost:5500`

## Purpose

- Validate we can keep your visual style outside HugoBlox.
- Build confidence before migrating the personal website.
